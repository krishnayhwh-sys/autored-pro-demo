# AutoRed Pro — Demo

Prototipo visual **navegable** de AutoRed Pro, una plataforma para conectar dueños de vehículos con
talleres automotrices certificados.

- 🖱️ **Prototipo demostrativo:** 14 pantallas navegables (cliente, taller y administrador).
- 🧪 **Datos completamente simulados:** nombres, talleres, precios y cifras son ficticios.
- 🚧 **No es todavía una aplicación productiva:** no hay backend, cuentas, pagos ni datos reales.

## Ver el demo

Abre `index.html` en cualquier navegador, o visita la versión publicada con GitHub Pages.
